   41: =head1 DESCRIPTION
   42: 
   43: This module provides a single function, C<datecmp>, which compares two date strings
   44: or date-like objects, returning a numeric comparison similar to Perl's spaceship operator (C<< <=> >>).
   45: 
   46: The comparison is tolerant of approximate dates (e.g., "Abt. 1902", "BET 1830 AND 1832", "Oct/Nov/Dec 1950"),
   47: partial dates (years only), and strings with common genealogy-style formats.
   48: It attempts to normalize and parse these into comparable values using L<DateTime::Format::Genealogy>.
   49: 
   50: =head1 FUNCTIONS
   51: 
   52: =head2 datecmp
   53: 
   54:   my $result = datecmp($left, $right);
   55:   my $result = datecmp($left, $right, \&complain);
   56: 
   57: Compares two date strings or date-like objects and returns:
   58: 
   59: =over 4
   60: 
   61: =item * -1 if C<$left> is earlier than C<$right>
   62: 
   63: =item * 0 if they are equivalent
   64: 
   65: =item * 1 if C<$left> is later than C<$right>
   66: 
   67: =back
   68: 
   69: Parameters:
   70: 
   71: =over 4
   72: 
   73: =item C<$left>, C<$right>
   74: 
   75: The values to compare. These may be strings in a variety of genealogical or ISO-style formats,
   76: or blessed objects that implement a C<date()> method returning a date string.
   77: 
   78: =item C<$complain> (optional)
   79: 
   80: A coderef that will be called with diagnostic messages when ambiguous or unexpected conditions are encountered,
   81: e.g. when comparing a range with equal endpoints.
   82: 
   83: =back
   84: 
   85: =head1 SUPPORTED FORMATS
   86: 
   87: The function supports a variety of partial or approximate formats,
   88: including:
   89: 
   90: =over 4
   91: 
   92: =item * Exact dates (e.g. C<1941-08-02>, C<5/27/1872>)
   93: 
   94: =item * Years only (e.g. C<1828>)
   95: 
   96: =item * Approximate dates (e.g. C<Abt. 1802>, C<ca. 1802>, C<1802 ?>)
   97: 
   98: =item * Date ranges (e.g. C<1802-1803>, C<BET 1830 AND 1832>)
   99: 
  100: =item * Month ranges (e.g. C<Oct/Nov/Dec 1950>)

  102: =item * Qualifiers like C<BEF>, C<AFT>
  103: 
  104: =back
