   41: =head1 DESCRIPTION
   42: 
   43: This module provides a single function, C<datecmp>, which compares two date strings
   44: or date-like objects, returning a numeric comparison similar to Perl's spaceship operator (C<< <=> >>).
   45: 
   46: The comparison is tolerant of approximate dates (e.g., "Abt. 1902", "BET 1830 AND 1832", "Oct/Nov/Dec 1950"),
   47: partial dates (years only), and strings with common genealogy-style formats.
   48: It attempts to normalize and parse these into comparable values using L<DateTime::Format::Genealogy>.
   49: 
   50: =head1 FUNCTIONS
   51: 
   52: =head2 datecmp
   53: 
   54: =head3 Purpose
   55: 
   56: Compare two genealogy-style date strings or date-like objects, returning a
   57: numeric result equivalent to Perl's spaceship operator (C<< <=> >>).
   58: 
   59: =head3 Arguments
   60: 
   61: =over 4
   62: 
   63: =item C<$left> (required)
   64: 
   65: The left-hand date to compare. Accepts any of:
   66: 
   67: =over 8
   68:

   85:

   87: 
   88: =head3 Returns
   89: 
   90: =over 4

   92: =item * C<-1> if C<$left> is earlier than C<$right>

   99: 
  100: When either argument is C<undef>, prints a diagnostic to STDERR and returns C<0>.

  108: ambiguous conditions.
  109: 
  110: =head3 Usage Example
  111: 
  112:   use Date::Cmp qw(datecmp);
  113: 
  114:   my $cmp = datecmp('BET 1830 AND 1832', '1831');   # 0
  115:   my $cmp = datecmp('Abt. 1850', '1855');            # -1
  116:   my $cmp = datecmp('1 Jan 1996', '1996-2000',
  117:       sub { warn "ambiguous: @_" });
